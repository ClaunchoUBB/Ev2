## Explicación de cada clase

#### Cotización

Representa las cotizaciones, es decir, un conjunto de items con su respectivo valor


#### Item

Representa a un único mueble con múltiples o ninguna variante

#### Mueble

Representa literalmente un mueble

#### Variante

Representa literalmente una variante

#### Venta

Representa una cotización que ya se ha confirmado
